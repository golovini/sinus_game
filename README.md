# Running Sinusoid

A graphical application that displays an animated sine wave.

## Running

```bash
pip install pygame
python3 sinus_game.py
```

## Controls

| Key | Action |
|-----|--------|
| `↑` / `↓` | Increase / decrease amplitude |
| `←` / `→` | Narrow / widen (change frequency) |
| `c` | Speed up |
| `z` | Slow down (keep pressing to reverse direction) |
| `q` | Quit |

## Requirements

- Python 3
- pygame 2.x
